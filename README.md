## 진행 방식
- 발표는 매 주 토요일 오전 10시에 온라인 화상회의로 진행됩니다.
- 매 주 공부할 범위를 정하고 해당 범위를 학습한 뒤 각자의 편한 공간(블로그, 노션 등등)에 정리한다음 발표를 한다음 git 저장소에 커밋하는 방식으로 진행합니다.
- 자신이 발표하지 않은 범위를 학습하다 작은 궁금증이나 질문이 있는 경우에는 Issue에 남겨 기록하도록 합니다.
